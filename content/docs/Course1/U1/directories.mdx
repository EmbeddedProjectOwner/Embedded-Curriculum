## Files/Directories
A file path, is referred to as a directory (dir). 

There are two types of directories in a typical file system:


- ### Absolute Directories
Absolute Directories are `Complete`, they are the most specific way of defining a file path within a filesystem.

Here's an example. (Using a Windows computer directory)

`C:\Users\User\ProjectsDirectory\Embedded-Curriculum\todo.md`
In this example:
- Root Filesystem: `C:\ (The C: drive, or boot drive)`

- ### Relative Directories

A relative directory is 'relative', to the base file system. What this means is, the file path is relative to the file that refrences it.

Relative directories use a `.` or `..` or sometimes custom aliases to define the relative directory.

Using `./` will define the parent of whatever file you're in.
